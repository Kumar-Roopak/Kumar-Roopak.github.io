What is AJAX

  1. Asynchrouous JavaScript & XML
  2. Set of web technologies
  3. send & recive data asynchronsly
  4. Does not interfere with the current page
  5. JSON has replaced XML for the most part
  6. Make asyc request in the background
  7. No page reload/refresh
  8. Fetch data
  9. Very interactive

XmlHttpRequest(XHR) object

  1. API in the form of an object
  2. Provided by the browsers JS environment
  3. Methods transfer data between client/ server
  4. can be used with other protocols than HTTP
  5. Can work with data other than XML (JSON, plain text)
